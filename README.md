# My Dotfiles for Arch Linux
## **Overview**

**Window manager:** i3-gaps

**Status bar:** i3bar with i3blocks

**Power manager:** xfce4-power-manager

**Image viewing/Background:** feh

**Screen Locker:** xscreensaver

**Notification manager:** dunst

**File manager:** ranger

**Text editor:** neovim

**Compositor:** compton  

**AUR helper:** yay

**Clipboard:** xsel

**Launcher:** rofi

**Shell:** bash
